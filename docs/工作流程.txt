git status / 查看更新狀態

git pull origin main / 從main上面取得最新版本

git checkout -b feature-new-ui / 創建分支, 避免直接在主線搞砸

......

開始寫代碼

代碼完成後

......

git add . / 此資料夾內所有有更動的元素都放入暫存區

git commit -m "xxx" / 建立永久存檔點, 'xxx'為本次更動的摘要(存檔點名稱)

git push origin main / 正式推到主線上更新。 origin: 這是遠端倉庫（GitHub）的代號。 / main: 你要把箱子寄到雲端的哪一條主線（分支）。

......

導入成功後

......

git pull origin main / 從main上面取得最新版本
